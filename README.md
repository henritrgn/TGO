# General Description : 

- This mod uses the random librairy to create randomized drops in minecraft with new items (you can use the items by eating them).

- 1.20.1 version

/!\ Loots can be found in folder : ...\TGO-master\src\main\java\net\henri\henritgo202423\item\custom

# How is structured this project ?
A minecraft mod is based on files that are located in "src" folder :
- There are elements about texture, sound, etc. ("resources" folder) ;
- There are elements to encapsulate the mod (the folder that has the mod name, here : "henritgo202423").

The rest of the stuff is basically gradle attributes to run in-game the mod (the build system).
